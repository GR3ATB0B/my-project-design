# Tailwind Color Configuration

## Color Choices

- **Primary:** `sky` — Used for buttons, links, key accents, active states
- **Secondary:** `violet` — Used for tags, highlights, secondary elements, avatar fallbacks
- **Neutral:** `slate` — Used for backgrounds, text, borders

## Usage Examples

Primary button: `bg-sky-500 hover:bg-sky-600 text-white`
Primary text: `text-sky-600 dark:text-sky-400`
Secondary badge: `bg-violet-100 text-violet-800 dark:bg-violet-900 dark:text-violet-300`
Neutral text: `text-slate-600 dark:text-slate-400`
Neutral border: `border-slate-200 dark:border-slate-800`
Neutral background: `bg-slate-50 dark:bg-slate-950`

## Additional Accent Colors

- **Emerald** — Used for success states, online status indicators, completed tasks
- **Amber** — Used for warning states, needs attention, error connections
- **Rose** — Used for high priority, failed states, destructive actions
